# Pihole with Proxmox

Setting up Pihole on Proxmox/Debian Linux.

## Install Docker

Just refer to the Docker website.

## Setup Pihole

- Use the run script [docker-run.sh](docker-run.sh) to setup Pihole.

- Replace ServerIP with your IP.

- And run it.

```bash
sh docker-run.sh
```
